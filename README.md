# Sourcier Québec — Site vitrine

Site vitrine de **Thomas, sourcier professionnel** basé à La Pocatière (Québec).

## Stack

- **Framework** : Astro (static site generator)
- **CMS** : Decap CMS (ex-Netlify CMS) — édition via interface web `/admin`
- **Hébergement** : Cloudflare Pages ou Netlify (à confirmer)
- **Source** : ce dépôt Git

## Objectif

Livrer un site à Thomas qu'il puisse éditer seul sans compétences techniques :
- Textes, prix, coordonnées modifiables via interface Decap
- Ajout/édition de FAQ et forfaits
- Publication automatique au commit

## Structure des sections

1. Héro (slogan + accroche)
2. Les 3 Piliers (Précision / Économie / Aide à la décision)
3. À Propos & Services
4. Forfaits (Résidentiel 250 $ / Agricole 350 $ / Municipal 550 $)
5. Zone d'intervention (Chaudière-Appalaches, Bas-Saint-Laurent)
6. FAQ
7. Témoignages (Google Business Profile)
8. Formulaire de contact
9. Footer + mentions légales + clause de non-garantie
